Perguntas Frequentes

1. Starbound Rank é gratuito?

Não. Há uma cobrança única para as empresas que desejam implementar o sistema, com planos personalizados de acordo com o porte e necessidades.

2. Preciso instalar algo?

Sim! Starbound Rank necessita do aplicativo instalado em dispositivos móveis (iOS/Android) e acesso web para gestores via navegador.

3. O sistema é seguro?

Sim, todas as informações são criptografadas com padrão AES-256 e armazenadas em servidores com certificação ISO 27001.

4. As recompensas de rank para o salário se acumulam?

Não, a cada novo rank atingido elas se atualizam conforme a tabela de progressão, substituindo o benefício anterior.
